Node is a standalone application for running scripts written in JavaScript. Node scripts are run from a terminal command line, just like shell scripts and Python or Ruby programs. The React CLI (command line interface) is built on top of Node.
Node includes many functions not available in browser JavaScript, including funtions to read and write files, write messages to the terminal, create processes, and so on.
Because Node does not run in a browser, it does not have access to a DOM (document object model), a browser console or debugger, or methods such as window.location.
There are tens of thousands of library packages developed for Node programming. npm, the Node Package manager, takes care of installing the packages a Node application needs.
Express is a popular web server written in Node.
