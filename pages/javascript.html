JavaScript has changed a great deal in the past five years. EcmaScript 2015 and later have added a large number of important extensions, many of which React depends on, including
  • Import and export
  • Classes
  • Arrow functions
  • Promises and async/await
  • Destructuring assignment
When reading React tutorials, or adapting React code, the JavaScript decoder may be helpful.
In addition, functional programming is increasingly emphasized in modern JavaScript. Many changes to the React libraries have been to support more functional code for global state and event handling.
