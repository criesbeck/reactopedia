JavaScript has changed a great deal in the past five years. Underneath it's the same prototype-based language with full support for functional programming, but EcmaScript 2015 and later have added a large number of important extensions, many of which React depends on.
The most critical of these extensions, for React developers, are
  • Import and export
  • Classes
  • Arrow functions
  • Destructuring assignment
When reading React tutorials, or adapting React code, the JavaScript decoder may be helpful.
