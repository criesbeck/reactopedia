This is a centralized resource for students learning React and the React ecosystem.
Understanding React requires understanding a large ecosystem of technologies.
These include modern JavaScript, i.e., EcmaScript 2015 (also known as ES6), NodeJS (also known as just Node), npm (the Node Package Manager), React itself, the Google Chrome debugger with the React extension, ReactRouter for connecting page-level components with URL pattern, and Redux for managing data that needs to be shared between components.
